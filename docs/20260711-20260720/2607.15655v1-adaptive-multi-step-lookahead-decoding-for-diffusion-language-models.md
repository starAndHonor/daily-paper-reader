## 摘要
掩码扩散语言模型通过迭代优化掩码令牌实现并行文本生成，为自回归解码提供了一种有前景的替代方案。最近基于前瞻的解码方法通过在提交令牌更新前探索未来解码状态，改进了精度与效率的权衡。然而，现有方法主要依赖浅层的一步前瞻，这优化了即时信息增益，但对于更长周期的解码轨迹可能并非最优。同时，我们发现简单扩展至更深的前瞻也无效，因为固定深度的展开会引入额外计算，且无法适应异构的中间解码状态。因此，本文提出AdaLook，一种用于扩散语言模型解码的自适应前瞻框架。AdaLook基于候选分数方差动态决定是否继续展开，并在中间展开状态需要额外探索时启用分支扩展。这种设计避免了不必要的深度展开，同时允许解码器从信息丰富的中间状态重新触发前瞻。在多种基准和模型上的实验表明，与现有的一步前瞻解码方法相比，AdaLook实现了更好的精度-解码步数权衡。

## Abstract
Masked diffusion language models (DLMs) enable parallel text generation by iteratively refining masked tokens, offering a promising alternative to autoregressive decoding. Recent lookahead-based decoding methods improve the accuracy--efficiency trade-off by exploring future decoding states before committing token updates. However, existing approaches mainly rely on shallow one-step lookahead, which optimizes immediate information gain but can be suboptimal for longer-horizon decoding trajectories. Meanwhile, we find that a naive extension for deeper lookahead is also ineffective, as fixed-depth rollout introduces additional computation and cannot adapt to heterogeneous intermediate decoding states. Thus, in this work, we propose AdaLook, an adaptive lookahead framework for DLM decoding. AdaLook dynamically determines whether to continue rollout based on candidate-score variance and further enables branch expansion when intermediate rollout states require additional exploration. This design avoids unnecessary deep rollout while allowing the decoder to re-trigger lookahead from informative intermediate states. Experiments on various benchmarks and models demonstrate that AdaLook achieves a better accuracy--decoding steps trade-off than existing one-step lookahead decoding methods.

## 1. 核心问题与整体含义（研究动机和背景）

扩散语言模型（DLM）通过迭代精炼掩码令牌实现并行文本生成，具有比自回归模型更灵活的推理潜力。然而，其去噪训练目标并未规定具体的解码顺序或令牌选择策略，留下了大量推理时设计空间。现有高效解码策略中，基于前瞻（lookahead）的方法（如Fu et al.的Explore-then-Exploit, ETE）通过评估未来解码状态来选择令牌更新，但仅使用**单步**前瞻，优化的是即时信息增益，对于更长周期的解码轨迹可能次优。同时，简单地增加前瞻步数（固定深度 rollout）会引入额外计算开销，且无法适应不同例子的异构中间解码状态。因此，论文旨在提出一种**自适应多步前瞻**框架，动态决定展开深度并允许分支扩展，以在精度与解码步数之间实现更好的权衡。

## 2. 方法论

### 核心思想
构建在ETE框架之上，扩展为自适应多步前瞻。ETE由**快速块扩散采样**（Fast Block Diffusion Sampling）和**置信度驱动的单步前瞻机制**组成。AdaLook在此基础上引入两个关键组件：

- **自适应展开延续（Adaptive Rollout Continuation）**：保留k个候选解码轨迹，每步计算累计展开评分（公式1），并监控候选分数方差。若方差低于阈值τ，表示候选不够区分，则继续展开；否则，若方差≥τ或达到最大深度Tmax，则提交最佳候选。
- **动态分支扩展（Dynamic Branch Expansion）**：每次展开后，独立检查每个分支是否需重新触发前瞻（基于ETE触发条件）。分三种情况处理：
    - 无分支需扩展：继续自适应展开。
    - 全部分支需扩展：选择评分最高（结合最新前向置信度）的候选，结束当前轮次，将其状态作为下一轮前瞻的起点。
    - 混合状态：剪枝仍需扩展的分支，仅保留稳定分支继续迭代。

### 关键技术细节
- 累计评分公式（公式1）：  
  \(\tilde{s}^{(R)}(j) = \alpha \log c_j^t(x_t) + \log\left(\sum_{r=1}^R \sum_{i \in T_j^r} c_i^{t+r}(x_{t+r;j})\right)\)  
  其中R为当前展开深度，T_j^r是在第r步变为高置信度并被提交的位置集合。
- 自适应判断：利用候选分数方差（Var>τ）决定是否继续展开；方差阈值τ和最大深度Tmax在实验中固定（k=4, α=0.1, β=0.01, Tmax=2, cinfo=0.2）。
- 超参数选择策略：分组处理，部分固定，部分通过小规模校准集确定（如置信度阈值C与块解锁预算N之间的对数关系），部分灵活设置（如γ在[0.1,0.4]中选优）。

### 算法流程（文字描述）
1. 从当前状态选择k个候选位置（基于置信度接近目标探索水平cinfo和位置偏置）。
2. 对每个候选执行一步前向，计算初始评分。
3. 若候选分数方差<τ，进入多步展开循环（最多M-1步）：
   - 每个分支提交高置信令牌并执行前向。
   - 检查每个分支是否需要重新触发前瞻。
   - 根据三种情况处理：全触发则选最优并返回；无触发则继续积累评分并检查方差；混合触发则保留稳定分支，继续迭代。
4. 方差达到阈值或循环结束，提交最佳候选。
